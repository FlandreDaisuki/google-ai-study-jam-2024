## 小技巧

- 如果有 `Start Lab` 的環節，請務必在開始之後，右鍵連結按鈕使用無痕模式開啟
- 當右上角的進度為 100/100 即可點 `End Lab`，如果發現所有綠色圈圈都已經勾選但進度沒有變成 100/100，可以多點幾次進度開開關關，他會更新
- 比較簡單的課程只要做 Quiz 即可拿到 Badge
